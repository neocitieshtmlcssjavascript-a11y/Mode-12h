# DOS Game in VGA Mode 12h (640x480, 16 Colors)

Проект старой школы (Retro Gaming) — заготовка игры на языке C для операционной системы MS-DOS. Использует графический режим VGA **Mode 12h** с разрешением 640x480 пикселей и палитрой из 16 цветов.

## 🚀 Требования для запуска

Так как это 16-битное DOS-приложение, оно не запустится напрямую на современных 64-битных Windows, macOS или Linux. Вам понадобятся:

1. **Эмулятор**: [DOSBox](https://dosbox.com) (или DOSBox-X / DOSBox-Staging).
2. **Компилятор**: Turbo C++ 3.0, Borland C++ 3.1 или Open Watcom (для DOS).

## 🛠️ Инструкция по сборке и запуску (через DOSBox)

### Шаг 1: Подготовка папок
1. Создайте на вашем компьютере рабочую папку, например: `C:\dosprojects\`.
2. Распакуйте туда ваш компилятор (например, Turbo C++ в папку `C:\dosprojects\TC\`).
3. Поместите файлы игры (исходный код `.c`) в папку `C:\dosprojects\game\`.

### Шаг 2: Запуск DOSBox и монтирование
Откройте DOSBox и введите команды для подключения вашей папки как диска `C:` inside DOSBox:

```bash
mount c c:\dosprojects
c:
```

### Шаг 3: Компиляция в Turbo C++
1. Запустите компилятор внутри DOSBox:
   ```bash
   cd tc\bin
   tc.exe
   ```
2. В меню Turbo C++ выберите `File -> Open` и откройте ваш файл `..\..\game\main.c`.
3. Нажмите `Ctrl + F9` для компиляции и автоматического запуска игры.

### Шаг 4: Прямой запуск скомпилированной игры
После успешной компиляции вы можете запускать готовый `.exe` файл прямо из консоли DOSBox:
```bash
cd \game
main.exe
```

## 📐 Особенности видеорежима Mode 12h

- **Разрешение**: 640x480 пикселей (соотношение сторон 4:3).
- **Цвета**: 16 фиксированных цветов EGA/VGA палитры.
- **Память**: Использует планарную структуру памяти (4 битовые плоскости). Прямой доступ к памяти `0xA000:0000` требует переключения регистров контроллера VGA через порты `0x3CE` и `0x3C4`.
- **Сегмент видеопамяти**: `0xA000` (стандартный для графики VGA).
